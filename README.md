# react-native-qiniu-uploader

## 功能介绍

- 基于七牛云存储，封装支持断点续传的React-Native控件，支持 iOS Android

## 安装使用

 `npm install --save react-native-qiniu-uploader`

Then link with:

 `react-native link react-native-qiniu-uploader`
